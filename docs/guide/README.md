## 安装

1. 安装[Tampermonkey](https://www.tampermonkey.net/)扩展
2. 安装[auto-task](https://auto-task.hclonely.com/auto-task.user.js)脚本

## 使用

### 其乐论坛

1. 在其乐论坛福利放送版块的帖子中的社交平台链接会添加相关功能
2. 左键单击就会执行相应的任务
3. 右键可以多选任务，然后点击"FuckTask"按钮批量完成

![Example](/keylol.gif)

### 非其乐论坛

1. 打开[支持的网站](#支持的网站及任务)的赠 Key 页面
2. 页面加载完成后右上角显示 1-4 个蓝色按钮（按钮数量不同网站可能不同）
3. 点击`FuckTask`一键完成[支持的任务](#支持的网站及任务)，点击`Remove`撤销之前的任务

> ps. 各按钮的具体功能及说明可在[设置页面](https://auto-task.hclonely.com/setting.html)自定义配置
