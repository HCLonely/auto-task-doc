## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) extension
2. Install [auto-task](https://auto-task.hclonely.com/auto-task.user.js) script

## Usage

### Keylol Forum

1. Relevant functions will be added to the social platform links in the posts in the welfare section of the Keylol Forum
2. Left click will execute the corresponding task
3. Right click to select multiple tasks, and then click the "FuckTask" button to complete batches

![Example](/keylol.gif)

### Except Keylol Forum

1. Open the giveaway page of [supported sites](#supported-sites-and-tasks)
2. After the page is loaded, 1-4 blue buttons are displayed in the upper right corner (the number of buttons may be different for different websites)
3. Click `FuckTask` to complete [supported tasks](#supported-sites-and-tasks), click `Remove` to cancel the previous task

> ps. The specific function and description of each button can be customized in [Settings Page](https://auto-task.hclonely.com/setting.html)
